#### To use OTP authentication through Email
- create an App Password of gmail account
    - Go to: https://myaccount.google.com/security
    - Enable 2-Step Verification
    - Then go to: https://myaccount.google.com/apppasswords
    - Create:
        - App → Mail
        - Device → Windows (or Other)
- then create this variables:
    - EMAIL_HOST_USER = "person1@gmail.com"
    - EMAIL_HOST_PASSWORD = "adfg ertt wedf edag"

- I have created one account which is :

    - password = person1@123
    - email = person1@gmail.com
